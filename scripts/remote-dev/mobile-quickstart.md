## Essential Commands (via Command Palette)

| Type this... | To do this... |
|-------------|---------------|
| `composer` | Open AI Composer |
| `chat` | Open AI Chat |
| `terminal` | New Terminal |
| `file` | Go to File (quick open) |
| `save` | Save current file |
| `git` | Git commands |
| `format` | Format document |

```

```

### Bug Fix
```
Fix the bug in [file path] where [bug description].
The error message is: [error message].
```

### Refactoring
```
Refactor [file/function] to [improvement].
Maintain backward compatibility.
```

### Tests
```
Write unit tests for [file/module].
Cover edge cases and error conditions.
```

### Code Review
```
Review [file path] and suggest improvements for:
- Performance
- Readability
- Error handling
```

## Virtual Keyboard

If you need special keys (`Esc`, `Ctrl`, `Tab`):

1. Look for floating circle/bar at screen bottom
2. Tap to reveal modifier keys
3. Tap modifier, then tap letter key

---

## Quick Troubleshooting

| Problem | Try This |
|---------|----------|
| Disconnected | Reopen PWA, reconnect |
| Can't type | Tap inside editor area |
| AI not responding | Check desktop is awake |
| Keys not working | Use Command Palette instead |
