# Product Roadmap

## Phase 1: MVP (Current)

- Authentication (signup, login, session management)
- World management (create, switch, delete worlds)
- World Bible elements:
  - Characters (CRUD, detail views, forms)
  - Locations (hierarchy support)
  - Organizations
  - Timeline/Events
  - Items
  - Rules
  - Stories/Scenes/Shots
- Dashboard with element counts
- Export page (placeholder)
- Chat page (placeholder)

## Phase 2: AI Integration

- AI World Chat with context awareness
- Story suggestions based on world data
- Consistency checking against defined rules
- Character/location relationship suggestions

## Phase 3: Production Export

- MetaHuman character specs export
- Environment asset requirements export
- Storyboard generation from shots
- Scene breakdown reports

## Phase 4: Collaboration

- Team workspaces
- Shared worlds
- Comments and annotations
- Version history
