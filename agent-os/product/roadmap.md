# Product Roadmap

## Phase 1: MVP

### World Bible Core
- 9 element categories: Characters, Locations, Organizations, Events, Items, Rules, Stories, Scenes, Shots
- Relationship mapping between elements
- Search and filter across all elements

### AI Features
- **Import/Generate World Bible** - Upload existing documents (PDF, DOCX, TXT), AI extracts and structures data into Vixio format
- **AI World Chat** - Query your world, get suggestions, check consistency
- **Rule Enforcement** - Define world rules, AI catches contradictions

### Export Formats
- Screenplay format
- Script format
- Production specs (MetaHuman character briefs, environment briefs, storyboards)

### Infrastructure
- User authentication (Supabase Auth)
- Multi-world support (switch between projects)
- Real-time sync
