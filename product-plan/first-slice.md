# First Implementation Slice

## Goal

Prove the Studio pivot with one coherent loop:

`brief or script -> intake -> boards/canon/assets framing -> export-ready project context`

## Scope

### User Entry

- project brief path
- script breakdown path
- reference intake path

### Workspace Surfaces

- Overview
- Boards
- Canon
- Assets
- Exports
- Agent Chat

### Reused Backing Data

- stories
- scenes
- shots
- characters
- locations
- organizations
- items
- rules
- events

## What This Slice Proves

1. The product identity has changed
2. Navigation is workflow-first
3. Existing data can support the new model immediately
4. Import now behaves like intake, not the whole product
5. Export is positioned as the final stage of one workflow

## What This Slice Does Not Yet Prove

- full board lineage
- true element-level refinement
- full multi-agent orchestration
- rich storyboard generation

Those belong to later execution phases, but this slice establishes the right foundation.
