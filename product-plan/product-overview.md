# Vixio Worldbuilder - Product Overview

## What It Is

Vixio Worldbuilder is a cloud-based worldbuilding tool for filmmakers and storytellers creating virtual productions with Unreal Engine.

## Target User

Solo creators and small teams working on short films, indie features, and narrative projects.

## Core Value

Bridge the gap between creative worldbuilding and production-ready specs for virtual production pipelines.

## Key Features

1. **World Bible** - 9 element categories (Characters, Locations, Organizations, Events, Items, Rules, Stories, Scenes, Shots)
2. **AI World Chat** - Query, consistency checking, story suggestions
3. **Rule Enforcement** - Define world rules and catch contradictions
4. **Production Pipeline** - Export specs for MetaHuman, environments, storyboards

## Problems Solved

| Problem | Solution |
|---------|----------|
| World details scattered | Unified, structured database |
| Inconsistencies in large worlds | Automatic rule checking |
| Gap between worldbuilding and production | Export production-ready specs |
| Hard to find story opportunities | AI suggests connections |

## Tech Stack

- Frontend: React + TypeScript + Tailwind CSS v4
- Backend: Supabase (PostgreSQL, Auth, Realtime)
- Deployment: Railway.app
- Target: Unreal Engine 5 virtual production

## Brand Identity

- **Logo**: Triangular mark with cyan-to-cream gradient
- **Colors**: Sky (primary), Teal (secondary), Slate (neutral)
- **Typography**: Space Grotesk, Inter, JetBrains Mono
