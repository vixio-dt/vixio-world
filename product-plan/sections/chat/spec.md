# Chat Section Specification

## Overview

The Chat section provides an AI-powered interface for interacting with the world. Users can query world information, get story suggestions, check consistency, and discover connections between elements.

---

## User Stories

### As a worldbuilder, I want to:
1. Ask questions about my world in natural language
2. Check if new ideas conflict with existing rules
3. Get suggestions for stories based on my world elements
4. Find underdeveloped areas that need attention
5. Discover connections I hadn't noticed

## AI Response Types

### Query Response
- Answer to the question
- Source citations (which elements informed the answer)
- Links to related elements

### Consistency Check
- Result: consistent | warning | conflict
- Applicable rules cited
- Suggestions for resolution

### Story Suggestion
- Story title and logline
- Characters that could be involved
- Locations that would feature
- Themes and conflicts
- "Develop This" action button

### Gap Analysis
- List of underdeveloped areas
- Priority suggestions
- "Fill Gap" action for each
